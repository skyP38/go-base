## Проекты

### 1. mk_go
**Описание**: Простой RESTful API, созданный с использованием фреймворка Gin. Проект основан на официальном туториале [go.dev](https://go.dev/doc/tutorial/web-service-gin).  
**Технологии**: Go, Gin  
**Цель**: Освоение основ создания веб-сервисов на Go.

### 2. go_rest (URL Shortener)
**Описание**: Сервис для сокращения длинных URL-адресов. Позволяет пользователям преобразовывать длинные ссылки в короткие и удобные для использования.  
**Технологии**: Go  
**Цель**: Практика в построении RESTful API для решения конкретной прикладной задачи.

### 3. go_grpc
**Описание**: Сервер, реализующий систему единого входа (Single Sign-On, SSO) с использованием протокола gRPC.  
**Технологии**: Go, gRPC  
**Цель**: Изучение механизмов удаленного вызова процедур (RPC) и построение аутентификационных сервисов.

### 4. go_kafka
**Описание**: Проект для работы с Apache Kafka на языке Go. Включает примеры по интеграции и использованию Kafka в Go-приложениях.  
**Технологии**: Go, Apache Kafka  
**Цель**: Освоение принципов асинхронной обработки сообщений и потоковой передачи данных.

## Структура репозитория
Репозиторий организован по принципу одного проекта в каждой директории. Каждый проект является самостоятельным и может быть изучен или запущен независимо от других.
